# mydata
学习SpringBoot与各项技术的整合

主体框架:SpringBoot  
项目构建:Maven  
版本控制:Git  
数据库:MySQL  
ORM映射:MyBatis  
日志管理:Log4j  
系统缓存:Redis  
反向代理:Nginx  
应用容器:Tomcat集群
